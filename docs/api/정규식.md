| 필드         | 정규식                                                                                  | 의미/조건 요약                                                                                                                                                        | 허용 예시                                          |
| ---------- | ------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- |
| `loginId`  | `^[a-z0-9]{4,20}$`                                                                   | 영문 **소문자 + 숫자만** 사용 가능, 전체 길이 **4~20자**. 문자열 전체가 이 규칙을 따라야 함.                                                                                                   | `abcd`, `user1`, `test1234`, `a1b2c3`          |
| `password` | ```^(?=.*[A-Za-z])(?=.*\\d)(?=.*[~!@#$%^&*()_+\\-=?{}\\[\\]\|:;\"'<>,./]).{8,20}$``` | 영문, 숫자, 특수문자를 포함해 8~20자                                                                                                                                         | test1234!                                      |
| `phone`    | ```^01[016789]-?\d{3,4}-?\d{4}$```                                                   | 한국 휴대폰 번호 형식• `01[016789]` : 010, 011, 016, 017, 018, 019로 시작• 중간 자리 `\d{3,4}` : 3자리 또는 4자리 숫자• 마지막 `\d{4}` : 4자리 숫자• `-?` : 하이픈(`-`)은 있어도 되고 없어도 됨 (중간, 끝 둘 다) | `01012345678`, `010-1234-5678`, `011-234-5678` |
| `Gender`   | ^(M\|F\|OTHER\|UNKNOWN)$                                                             | M, F, OTHER, UNKNOWN 중 하나여야                                                                                                                                     | M, F, OTHER, UNKNOWN                           |
